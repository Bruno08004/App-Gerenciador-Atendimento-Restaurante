# App-Gerenciador-Atendimento-Restaurante
Este projeto foi desenvolvido como parte da disciplina de Estrutura de Dados, em conjunto com Linguagem de Programação 2, e tem como objetivo aplicar os principais conceitos de Programação Orientada a Objetos (POO) e o uso prático de estruturas de dados no contexto do desenvolvimento de um aplicativo de gerenciamento de restaurante.
